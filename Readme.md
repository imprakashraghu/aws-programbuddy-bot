# Program Buddy - Creating Project using ChatBot
### Send messages in WhatsApp and get your project started right away.

Services Used:

- Amazon Lex
- Amazon Lambda
- Amazon Dynamo DB
- Amazon Simple Email As Service
- Amazon Cloud Watch
- Amazon S3 Bucket
- Twilio
